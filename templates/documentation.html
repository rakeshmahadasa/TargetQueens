COVFEFE

COVFEFE is a web application to track the trending news about any topic and twitter feed of influential persons. This application also helps the users to find out the keywords and sentiment in the news articles as well as the tweets of the person, the user searched for.

This is part of the assigment by Dr. Hassan and Dr. Cor-Paul
Tech Stack : Python Libraries, Flask, HTML, Javascript, Jquery, Twitter API

Requirements

  • A crawler that crawls the latest 25 articles about Trump from CNN.com and his latest 25 tweets
  • A simple website that displays the titles of the crawled information
  • A convenient way of displaying the information after I click on one of the titles
  • A cool feature that I came up with yourself.

Deliverables

  • A API to search about any trending topic not just Trump
  • An option to configure the news from CNN, NYTimes, WS Journal
  • A convenient UI to display the news articles as well as latest 25 tweets.
  • A convenient UI to display the Title, Description, Summary, Keywords and Sentiment in the News Article, once the title is clicked
  • An Option to show the keywords from the twitter feed as well as the sentiment of the latest 25 tweets of the user.
